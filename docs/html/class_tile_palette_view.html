Detailed Description

Represents the view for the tile palette, and hence is responsible for how users perceive the notion of a tile palette.

Some references to kiss_sdl library, ideally to be separated.

An implementation of the view that uses kiss_sdl, a simple GUI library.

Note that this object is not responsible for deleting the given features.

Parameters
featuresthe high-level application events

As of now, there is one kiss window. To select a tile index, you have to enter a non-negative integer into the text box. We will trust the user to not plug in a negative value for now. In the future, there will be a new window, similar to the tilemap, but for the texture atlas.
