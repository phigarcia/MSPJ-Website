Detailed Description

This represents the notion of a spritesheet/texture atlas, i.e an image file that is actually composed of smaller textures, but has been put into one file so that they can be loaded into engine as one unit.

It is currently assumed that the smaller images all have the same width height, and have no gaps with each other.
