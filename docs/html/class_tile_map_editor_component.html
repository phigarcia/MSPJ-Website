Detailed Description

This is an extension for the game engine to allow users to edit the tile map with a paint brush and tile palette mechanism.

HandleKeyEvent allows for users to change to brush mode and edit mode with "B" and "E" respectively.

Users can modify the tile map by clicking on a particular tile, or clicking and dragging across many tiles.

The mouse handler events call PaintTileAtWindowPos to paint a tile at the world position the mouse is pointing at. That is how the logical notion and visual representative of the tile map is updated.
