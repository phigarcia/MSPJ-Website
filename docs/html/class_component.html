Detailed Description

An abstract base class for all components.

A Component must be attached to a GameObject. WARN: Do not manually instantiate a component via a constructor. Always create a component using Engine's InstantiateComponent<T> method.

WARN: when creating a new component, always write a virtual destructor, and mark override any desired virtual life cycle functions.
